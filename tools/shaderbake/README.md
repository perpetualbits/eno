# tools/shaderbake

GLSL shader minification and packing for size-coded productions.

## Planned features

- Parse GLSL, fold constants, inline single-use functions
- Rename identifiers to single characters
- Strip whitespace and comments
- Pack final shader strings into a C header for inclusion in the demo
- Optionally produce a SPIR-V variant for Vulkan-based productions
